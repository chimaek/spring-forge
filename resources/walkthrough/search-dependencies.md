## Search Dependencies

Press **/** anywhere in the panel to jump to the search bar.

| Key | Action |
|-----|--------|
| `/` | Focus search |
| `Arrow Up/Down` | Navigate results |
| `Enter` or `Space` | Toggle dependency |
| `Escape` | Return to search |

Search matches against **name**, **ID**, and **description** simultaneously — so typing `jpa` finds Spring Data JPA, and typing `web` finds Spring Web, WebFlux, and more.

Selected dependencies appear as **removable badges** above the list.
