## Open Spring Forge

There are three ways to open the project generator:

| Method | How |
|--------|-----|
| Activity Bar | Click the **Spring** icon in the left sidebar |
| Command Palette | `Ctrl+Shift+P` → `Spring: New Project (Initializr UI)` |
| Keyboard shortcut | `Ctrl+Alt+S` (Mac: `Cmd+Option+S`) |

The visual two-panel UI will open — project settings on the left, dependency search on the right.
