## Preview & Generate

### Preview
Click **Preview** to see the exact `pom.xml` or `build.gradle.kts` that will be generated — before creating anything.

This fetches the real build file from the Spring Initializr API, so what you see is what you get.

### Generate
Click **Generate** to:
1. Download the project ZIP from start.spring.io
2. Extract it to your chosen folder
3. Open it in VS Code (current window or new window)
4. Detect Maven/Gradle and suggest running the project
5. Recommend Java Extension Pack if not installed
