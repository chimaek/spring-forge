## Save a Preset

After configuring your project (groupId, language, build type, dependencies), click **Save Preset** in the header area.

Your preset stores:
- Build type (Maven / Gradle / Gradle Kotlin DSL)
- Language (Java / Kotlin / Groovy)
- Java version
- Group ID & Artifact ID
- All selected dependencies

Next time, load a preset from the dropdown to apply everything at once — no more repetitive configuration.
